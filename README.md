# To install
```
npm i -g @maurya-ankit/file-manager
```

above installation may require sudo access


# To use it further

```
$ file-manager
```
or if you want any manual port number (defaults to 7070)
```
$ file-manager -p PORT
```

open any browser and manage files and folder at `http://localhost:PORT`
